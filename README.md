### Prerequisites

Ruby: 2.7.2
Rails: 5.2.3
Postgres: >=13

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:seed
   rails db:migrate
```


## Use

- Navigate to the root directory of the project
- Run `rails server`
- Open a browser and visit `127.0.0.1:3000`
- Sign up and a create transactions with or without a group!
